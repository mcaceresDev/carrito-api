Primeros pasos

Estas instrucciones le proporcionarán una copia del proyecto en funcionamiento en su máquina local para fines de desarrollo y prueba. Consulte implementación para obtener notas sobre cómo implementar el proyecto en un sistema en vivo. Ve al repositorio del proyecto para clonarlo en tu computadora, seguido de esto ejecula los siguientes comandos desde la consola.

Instalación

  • Instala todas las dependencias
  • npm install
  • Crea un archivo .env
  • touch .env
  • Agrega una secret key a tu archivo .env
  • SECRET = ???

URL Base

La URL base es la URL raíz para toda la API, si alguna vez realiza una solicitud a carrito-api y obtiene un error, primero verifique la URL base.
La url base para carrito-api es:

http://localhost:3050/api/
