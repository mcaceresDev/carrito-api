Bienvenido a Carrito Api

Una api de ejemplo para app de ventas que incluye carrito de compras. Aquí encontrarás información suficiente para familiarizarte con la api y consumir sus recursos desde tu propia app, asi como el enlace al repositorio del código fuente del proyecto para que puedas personalizarlo a tu gusto y necesidades.

Acerca de

Carrito api fue desarrollada para ofrecer un recurso sencillo a partir del cual puedes partir para desarrollar tu propia app completa de e-commerce

Fue desarrollada con las siguientes tecnologias:

  • Node
  • Typescrip
  • Express
  • Sequelize
  • Dotenv
  • JsonWebToken

y cuenta con una base de datos MySql

Recursos

Carrito api cuenta con 4 recursos comunes:

  • /Productos
  • /Categorias
  • /Usuarios
  • /Ventas

Rutas

Todos los métodos HTTP son soportados:

  • GET /api/productos
  • GET /api/categorias
  • GET /api/usuarios
  • GET /api/ventas
